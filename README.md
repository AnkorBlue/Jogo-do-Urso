# Jogo do Urso

Este projeto recria o clássico jogo do dinossauro do Google, integrando-o com um **sensor ultrassônico conectado a uma placa Arduino**.  

O objetivo é unir **programação visual no Scratch** com **interação física via sensores**, proporcionando uma experiência **imersiva e educativa**.  

No jogo, o personagem principal **salta sempre que o jogador se aproxima do sensor**, simulando o movimento de pular no mundo real.
